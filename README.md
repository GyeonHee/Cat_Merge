## 1. 게임 메커니즘
  - 같은 등급의 고양이를 드래그 앤 드랍을 통해 합치기
  - 정해진 시간마다 등장하는 쥐를 잡기
## 2. 주요 목표
  - 더 높은 등급의 고양이를 아이템들을 활용하며 수집하여 성장하는 게임
## 3. 개발 환경
  - Unity, C#
